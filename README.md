# WELLCOME

##CLONE REPO

In order to clone this project type in terminal

```sh
$ git clone https://github.com/YaroslavAnt/statictic-app.git
```

Install the dependencies and devDependencies.

```sh
$ cd client
$ npm install
```

```sh
$ cd server
$ npm install
```

##SERVER

Init or update database

```sh
$ cd server
$ npm run init
```

Run server

```sh
$ cd server
$ npm run start
```

##CLIENT

Run application

```sh
$ cd client
$ npm run start
```
